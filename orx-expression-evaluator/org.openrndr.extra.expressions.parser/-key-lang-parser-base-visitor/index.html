KeyLangParserBaseVisitor

abstract class KeyLangParserBaseVisitor<T> : AbstractParseTreeVisitor<T> , KeyLangParserVisitor<T>

This class provides an empty implementation of KeyLangParserVisitor, which can be extended to create a visitor which only needs to handle a subset of the available methods.
