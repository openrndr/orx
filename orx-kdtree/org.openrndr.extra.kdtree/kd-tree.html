fun Iterable<Vector2>.kdTree(): KDTreeNode<Vector2>(source)

Constructs a KD-Tree for a collection of 2D vectors.

This function creates a KD-Tree from the given iterable collection of Vector2 objects. The KD-Tree is built in a way that organizes the points for efficient spatial operations, such as nearest neighbor search or range queries, considering two dimensions (x and y).

Return

The root node of the KD-Tree representing the input collection of 2D vectors.
