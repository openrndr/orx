fun SlidersVector2.bind(container: Any, property: KMutableProperty1<Any, Vector2>)(source)

Binds the SlidersVector2 component to a property within a container object. Updates are propagated between the slider’s value and the specified property, ensuring synchronization in both directions. The method listens to changes in the slider's value and modifies the container's property accordingly. Additionally, it ensures that the slider reflects the current value of the bound property.

Parameters

container

The object containing the property to be bound to the slider.

property

The mutable property within the container to bind with the slider's value.

Throws

RuntimeException

If the slider is not part of a Body.
