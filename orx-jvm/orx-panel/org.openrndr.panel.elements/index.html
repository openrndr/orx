fun SlidersVector2.bind(container: Any, property: KMutableProperty1<Any, Vector2>)

Binds the SlidersVector2 component to a property within a container object. Updates are propagated between the sliders value and the specified property, ensuring synchronization in both directions. The method listens to changes in the slider's value and modifies the container's property accordingly. Additionally, it ensures that the slider reflects the current value of the bound property.
