# orx-dnk3

A scene graph based 3d renderer with support for Gltf based assets

Status: in development

Supported Gltf features
- [x] Scene hierarchy
- [x] Loading mesh data
- [x] Glb
- [ ] Materials
  - [x] Basic materials
  - [x] Normal maps
  - [x] Metallic/roughness maps
  - [x] Skinning
  - [x] Double-sided materials
  - [ ] Transparency
- [x] Animations 
- [ ] Cameras
- [ ] Lights
